# 라이브 스트리밍 리스트 프로젝트

## 프로젝트 소개

라이브 스트리밍 리스트 프로젝트는 다양한 플랫폼에서 실시간 스트리밍 데이터를 수집하고 이를 사용자에게 제공하는 웹 애플리케이션입니다. 
이 프로젝트는 백엔드에서 데이터를 스크래핑하여 데이터베이스에 저장하고, 프론트엔드는 저장된 데이터를 기반으로 사용자에게 실시간 스트리밍 정보를 제공합니다.
